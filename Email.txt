Dear DistroWatch Team,

We are pleased to announce the official release of **Helwan Linux Dev v3.1**, a major update that represents a significant expansion of the Helwan Linux native ecosystem.

This release introduces a mature and integrated suite of **25+ native tools**, designed to deliver a clean, developer-centric experience while remaining fully compatible with the Arch-based ecosystem.

### Key Highlights

**Stable Base**
Helwan Linux Dev v3.1 is now powered by **Linux Kernel 6.12 LTS**, providing long-term stability, improved hardware compatibility, and reliable performance.

**Native Ecosystem Expansion**
Major enhancements have been made to core native tools such as **hpm**, **Momo**, and **Hel-Sync**, alongside the introduction of new specialized toolsets focused on:

* Security
* Development
* Daily productivity

**Enhanced Security**
This release includes critical security updates and addresses multiple vulnerabilities across core system components.

**Performance Improvements**
System-wide optimizations have been implemented to ensure a faster, smoother, and more stable desktop experience.

**New Native Toolsets**
Helwan Linux Dev v3.1 introduces several new utilities, including:

* **hel-sec-audit** for system security scanning
* **hel-builder** for development workflows
* A new category of **native Helwan Games**

> *Helwan Linux is an independent project focused on delivering a clean, developer-oriented environment. We believe this release provides meaningful value to the Arch-based Linux community.*
